# 🔐 MWR Cyber Security CTF – Ares Mainframe

## Overview
This project involved the security assessment of a simulated CTF environment, focusing on identifying and exploiting web application and system vulnerabilities.

---

## Scope
- Virtual CTF environment (TryHackMe)
- Web applications and internal services
- Authentication systems
- File handling mechanisms

---

## Tools Used
- Nmap
- Burp Suite
- Gobuster
- Exiftool
- John the Ripper
- CyberChef

---

## 📁 Project Structure

- Reconnaissance & Enumeration
- Web Application Exploitation
- File & System Exploitation
- Supporting Techniques

---

All testing followed a structured penetration testing methodology.
